### 🏆 Featured Project

#### ⌁ [Visual Pipeline Builder](https://github.com/Prabhat12112002/Pipeline-Builder)
A professional, enterprise-grade node-based workflow editor designed to build dynamic AI pipelines.
* **Frontend:** React, React Flow, Zustand, and Tailwind CSS. Features dynamic handle rendering and custom metadata-driven node layout abstractions.
* **Backend:** Python, FastAPI, and Pydantic. Implements an **iterative 3-color DFS cycle check** (DAG validation) to prevent recursion-limit crashes on complex graphs.
* **Key Innovation:** Custom text node variable parsed dynamically on input with real-time UI connection hook generation.
